# 文档绘图

使用文档绘制常见的数据结构并且解释思路

### 链表

```js
1 -> 2 -> 3
```

### 环形链表

```js
1 -> 2 -> 3
      |   |
       <--   
```

### 二叉树

```js
	 4
	/  \
   12   5
	     \
	     10
```

### 双指针

```js
  [1, 2, 3, 4, 5]
   ^           ^
 left        right
```

栈

```js
10
5
2
bottom
```

无向图

```js
1 - 4
|   |
2 - 3 - 10
```

有向图

```js
1 -> 4
|    |
v    v （字母 v）
2 <- 3 -> 10
```

前缀树

```js
     a
    p c
   p   t (end)
  l
 e (end)
 ```
